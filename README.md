## 几何球体
### 顶点绘制
几何球体和普通球体最大的不同就在于纹理排布
- 普通球体根据类似地球经纬度的方式排布顶点，纹理会有比较强烈的拉升
- 普通球体纹理能完整显示，集合球体纹理无法完全显示, 尽量合理调整
- 几何球体是由正十二面体作为原型, 将每一条弧切割成`n`份，最终绘制而成

### 纹理绘制
- 几何球体的纹理绘制是将几何球体的`正20面三角`形平铺在平面上后来绘制的。
